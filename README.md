mRepo
===============================

version number: 0.0.1
author: Yidong Ma

Overview
--------

A python package that can be installed with pip.

Installation / Usage
--------------------

To install use pip:

    $ pip install mrepo


Or clone the repo:

    $ git clone https://github.com/coder-james/mrepo.git
    $ python setup.py install
